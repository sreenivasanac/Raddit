My attempt of a Reddit-Like Website creating using Ruby on Rails after seeing some screencasts and RoR docs.

Users can SignUp, Login, Add Links, Upvote, Downvote and Comment.

I've used Twitter Bootstrap, devise gem, acts_as_votable gem
